# PyPI Deployment Guide

1. Create a PyPI account at https://pypi.org/
2. Update version in setup.py and docs
3. Build package:
   python -m build
4. Test with TestPyPI:
   python -m twine upload --repository testpypi dist/*
5. Install from TestPyPI:
   pip install --index-url https://test.pypi.org/simple/ recordflow
6. Deploy to PyPI:
   python -m twine upload dist/*
7. Verify install:
   pip install recordflow
8. Tag release and update CHANGELOG.md
